= イベント固有のデータ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
監査ログ内の各監査メッセージはシステムイベントに固有のデータを記録します。

メッセージ自体を識別する開いているコンテナに続く一連の属性は、 `[AUDT:`監査メッセージで説明されているイベントまたはアクションに関する情報を提供します。以下の例では、これらの属性を強調表示しています。
